# Wheat-Disease-Detection--YOLOv10
This project presents an AI-powered wheat leaf disease detection system using YOLOv10. The model accurately identifies and localizes wheat leaf diseases in real time, helping farmers monitor crop health and take timely preventive measures.

Features

* Real-time wheat disease detection
* Accurate disease classification and localization
* Web-based user interface
* Performance evaluation using precision, recall, and confusion matrix metrics
Technologies Used

* Python
* YOLOv10
* OpenCV
* Flask
* Deep Learning

Dataset

The model was trained on a wheat leaf disease dataset containing images of healthy and diseased wheat plants.

Results

The system successfully detects diseases such as:

* Leaf Rust
* Powdery Mildew
* Yellow Dwarf
* Healthy Wheat Leaves
* Performance visualizations included:

* Precision-Recall Curve
* Precision Curve
* Confusion Matrix
* Detection Output Images

Future Enhancements
